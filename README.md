#CALCULATOR

##Problem Description
A Program that can add, subtract, multiply, divide and cancel value

##Dev environment setup
- [Java OpenJDK 11](https://www.oracle.com/java/technologies/javase-jdk11-downloads.html)
- [Gradle](https://gradle.org/install/)

##Build instructions
`gradlew build`

##Run instructions
`gradlew run`

##Run test
`gradlew test`
